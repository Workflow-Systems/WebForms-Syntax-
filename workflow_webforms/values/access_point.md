# AccessPoint

## Шаблоны AccessPoint <a href="#template_access_point" id="template_access_point"></a>

```xml
<AccessPoint Name="" />
```

```xml
<AccessPoint Name="" DataConnection="Value" />
```

Указание имени соединения [DataConnection](../dataconnections/) необязательно, если точка доступа с заданным именем встречается только в одном из них.

В противном случае указание имени DataConnection обязательно.
