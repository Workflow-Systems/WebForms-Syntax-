# DataConnections

## Описание DataConnection <a href="#description_data_connection" id="description_data_connection"></a>

```xml
<DataConnection Name="DataConnectionName" Type="DataConnectionType" Assembly="DataConnectionAssembly">
  <!--Тэги, общие для всех соединений с данными-->
  <!--Тэги, специфичные для определенного соединения с данными (зависит от типа)-->
</DataConnection>
```
