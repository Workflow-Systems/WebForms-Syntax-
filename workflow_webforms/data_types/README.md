### Атрибуты DataType <a href="#attributes_data_type" id="attributes_data_type"></a>

<table data-header-hidden><thead><tr><th align="center"></th><th width="473.3333333333333"></th></tr></thead><tbody><tr><td align="center">Type</td><td><p>Название типа данных.</p><p></p><p>Обязательный атрибут.</p></td></tr></tbody></table>

Атрибуты типов данных делятся на 2 категории:

1. Для форматированного вывода значения.
2. Для проверки корректности задания значения типа.

### Типы данных поддерживают операции (используется в формулах - конструкция [Formula](../values/formula.md)): <a href="#data_type_operations" id="data_type_operations"></a>

1. Сложение.
2. Вычитание.
3. Умножение.
4. Деление.
